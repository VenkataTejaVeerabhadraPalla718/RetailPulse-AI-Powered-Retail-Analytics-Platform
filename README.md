# RetailPulse - AI Powered Retail Analytics Platform

## Project Overview

RetailPulse is an AI-powered retail analytics platform developed to help businesses make data-driven decisions using machine learning and business intelligence techniques.

The project provides:

- Customer Segmentation
- Customer Churn Prediction
- Demand Forecasting
- Inventory Optimization
- SHAP Explainability
- Interactive Streamlit Dashboard

---

## Features

### Customer Segmentation
- RFM Analysis
- K-Means Clustering
- Customer Group Identification

### Customer Churn Prediction
- Churn Detection Model
- Customer Retention Insights

### Demand Forecasting
- Prophet Time Series Forecasting
- Future Sales Prediction

### Inventory Optimization
- Reorder Quantity Calculation
- Risk Classification

### Explainable AI
- SHAP Summary Plot
- Feature Importance Analysis
- Waterfall Plot

### Dashboard
- Executive Summary
- Sales Analytics
- Customer Analytics
- Country Sales Dashboard
- Inventory Dashboard
- Forecast Dashboard
- SHAP Explainability Dashboard
